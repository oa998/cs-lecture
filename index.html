Life as a developer

At a company, the software is typically 10k - 100k lines of code per repository. Teams will often collaborate across many repositories.
Repository: a single collection of code that works together.
My team works across about 10 repositories. The entire project is probably 50 repositories.
Depending on the company, the code might be used by 10 people or 10 million.

Popular Questions

What's your favorite part of your job?

Designing the code to solve a problem. I usually do this with a notebook and pen.

Popular Questions

"What's hard about your job?

The complexities of how working with different teams all working on the same project. Sometimes you have to have one team develop part of the solution and another team develop a different part. These parts could each take months to develop and it has to be planned out really well to make sure it all works in the end.

Popular Questions

What's a work day like for you?
For me, I am responsible for 5 developers, so I actually spend a lot of time working with them to teach them. I teach them the correct ways to write code but also how all the different parts of the code work together.
When I approve a developer's code, I then allow it to be added to the software. ("merge")
I spend about 15 hours a week (at work) coding, but I spend my free time coding for fun, too.
Lastly, I work with the people above me to plan what new changes we need to make to the software and communicate that with my team.
